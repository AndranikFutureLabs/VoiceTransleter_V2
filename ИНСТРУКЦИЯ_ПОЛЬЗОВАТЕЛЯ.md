# Инструкция пользователя VoiceTransleter

## О программе

VoiceTransleter — программа для автоматического дубляжа видео с переводом английской речи в русскую. Программа распознаёт речь, переводит её и синтезирует голос на русском языке, сохраняя оригинальные тайминги.

## 1. Установка

### 1.1. Клонирование с GitHub

```bash
git clone https://github.com/AndranikFutureLabs/VoiceTransleter.git
cd VoiceTransleter
```

### 1.2. Установка зависимостей

Рекомендуется использовать виртуальное окружение:

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt
```

### 1.3. Установка FFmpeg

Скачайте FFmpeg с официального сайта: https://ffmpeg.org/download.html

Разместите файлы `ffmpeg.exe` и `ffprobe.exe` в папке `ffmpeg/` в корне проекта:

```
VoiceTransleter/
└── ffmpeg/
    ├── ffmpeg.exe
    └── ffprobe.exe
```

Либо добавьте папку с FFmpeg в системную переменную PATH.

### 1.4. Подготовка образца голоса (для XTTS)

Для использования качественного синтеза голоса XTTS необходимо поместить аудиообразец голоса в папку `voices/` с именем **Образец**.

Поддерживаемые форматы: **WAV, MP3, M4A, OGG**.

Полный путь к файлу должен выглядеть так:

```
C:\VoiceTransleter\voices\Образец.wav
```

или

```
C:\VoiceTransleter\voices\Образец.mp3
```

Рекомендации по выбору образца:
- Длительность: 5–15 секунд
- Чистая речь, без посторонних шумов и музыки
- Голос должен принадлежать человеку, чьим голосом вы хотите озвучить дубляж
- Формат WAV предпочтителен

### 1.5. Запуск программы

```bash
python main.py
```

## 2. Использование программы

### 2.1. Главное окно

После запуска открывается главное окно программы с тёмной темой.

### 2.2. Выбор видео

1. Нажмите на центральную область «Нажмите для выбора видеофайла»
2. В открывшемся диалоге выберите видеофайл
3. Поддерживаемые форматы: MP4, AVI, MKV, MOV, WebM
4. После выбора область загрузки станет зелёной, а кнопка «Начать дубляж» активной

### 2.3. Выбор голоса дубляжа

Из выпадающего списка выберите голос:
- **Silero** — встроенные русские голоса (работают сразу, без дополнительных загрузок)
- **XTTS** — высококачественный синтез голоса (требуется загрузка модели и образец голоса)

Если XTTS ещё не загружена, нажмите кнопку **«Загрузить XTTS»** (размер модели ~1.87 ГБ).

### 2.4. Прослушивание голоса

Нажмите **«Прослушать»**, чтобы услышать, как выбранный голос звучит. Будет синтезирована тестовая фраза.

### 2.5. Клонирование голоса

1. Нажмите **«Клонировать голос»**
2. Выберите аудиообразец (3–10 секунд)
3. Введите название для нового голоса
4. Нажмите **«Клонировать»** (процесс может занять несколько минут)
5. Новый голос появится в списке доступных голосов

### 2.6. Запуск дубляжа

1. Нажмите **«Начать дубляж»**
2. Программа последовательно выполнит 5 шагов:
   - **[1/5]** Извлечение аудио из видео
   - **[2/5]** Распознавание речи (Whisper)
   - **[3/5]** Перевод EN→RU
   - **[4/5]** Синтез речи (самый долгий шаг)
   - **[5/5]** Сборка финального видео
3. После завершения появится карточка с результатом

### 2.7. Результаты

После завершения дубляжа в папке `output/` создаются следующие файлы:

| Файл | Описание |
|------|----------|
| `*_dubbed.mp4` | Готовое видео с дубляжом |
| `*_source.txt` | Исходный текст с таймингами |
| `*_source_plain.txt` | Исходный текст без таймингов |
| `*_translation.txt` | Перевод с таймингами |
| `*_translation_plain.txt` | Перевод без таймингов |
| `*_source_translit.txt` | Транслитерация оригинала (En→Ru) с таймингами |
| `*_source_translit_plain.txt` | Транслитерация оригинала без таймингов |
| `*_translation_translit.txt` | Транслитерация перевода (Ru→La) с таймингами |
| `*_translation_translit_plain.txt` | Транслитерация перевода без таймингов |

Вы можете открыть готовое видео или папку с результатами, нажав соответствующие кнопки в карточке результата.

## 3. Структура папок

```
VoiceTransleter/
├── main.py                    # Графический интерфейс (Flet)
├── config.py                  # Конфигурационные параметры
├── requirements.txt           # Зависимости Python
├── ffmpeg/                    # Исполняемые файлы FFmpeg
├── voice_transleter/          # Основной модуль программы
│   ├── __init__.py
│   ├── pipeline.py            # Пайплайн (последовательность обработки)
│   ├── audio_extractor.py     # Извлечение аудио из видео
│   ├── transcription.py       # Распознавание речи (Whisper)
│   ├── translator.py          # Перевод текста
│   ├── tts.py                 # Синтез речи (Silero / XTTS)
│   ├── video_renderer.py      # Сборка финального видео
│   └── voice_cloner.py        # Клонирование голоса
├── voices/                    # Образцы голосов
│   └── Образец.wav           # Пример образца для XTTS
├── models/                    # Скачанные модели (создаётся автоматически)
├── temp/                      # Временные файлы (создаётся автоматически)
└── output/                    # Результаты дубляжа (создаётся автоматически)
```

## 4. Системные требования

- **ОС:** Windows 10/11, Linux, macOS
- **Python:** 3.10 или выше
- **ОЗУ:** 8 ГБ (рекомендуется 16 ГБ для XTTS)
- **Диск:** 5 ГБ свободного места (для моделей и временных файлов)
- **FFmpeg:** установленный и доступный в PATH

## 5. Часто задаваемые вопросы

**В: Программа не запускается, ошибка «FFmpeg не найден»**
О: Скачайте FFmpeg и поместите `ffmpeg.exe` и `ffprobe.exe` в папку `ffmpeg/` в корне проекта.

**В: Долго обрабатывается видео**
О: Самый долгий этап — синтез речи (XTTS). Для ускорения используйте Silero.

**В: XTTS не загружается**
О: Проверьте подключение к интернету. Модель весит ~1.87 ГБ. При ошибке попробуйте перезапустить программу.

**В: Как улучшить качество распознавания?**
О: Используйте видео с чистым звуком, без сильного фонового шума.

**В: Не сохраняются текстовые файлы**
О: Текстовые файлы сохраняются в папку с готовым видео (обычно `output/`).
